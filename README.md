# rn-expense-manager

Starter React Native (Expo) expense manager scaffold

## Install
`
cd rn-expense-manager
npm install
`

## Run
`
npm run start
`

## Structure
`
src/
  assets/
  theme/
  reducers/
  containers/
  components/
  navigators/
  common/
  utils/
`

## Nguyên tắc thiết kế
- Đơn giản & tập trung: Thêm giao dịch, xem tổng quan, theo dõi ngân sách
- Nhất quán: Màu sắc, khoảng cách, typography, trạng thái
- Phản hồi rõ ràng: Loading, empty-state, lỗi API, xác nhận thành công, undo
- Khả dụng: Cỡ chữ ≥14sp, tương phản đủ, dark mode, screen reader
- Mở rộng: Module hóa component, dễ thêm AI/Chatbot/OCR
